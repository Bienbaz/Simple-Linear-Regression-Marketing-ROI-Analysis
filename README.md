
# Simple Linear Regression – Marketing ROI Analysis

## Setup
pip install pandas numpy matplotlib seaborn statsmodels

## Files
- regression_analysis.ipynb
- marketing_and_sales_data_evaluate_lr.csv

## Workflow
1. Upload CSV file
2. Explore and clean data
3. Perform EDA
4. Select most correlated marketing channel
5. Build OLS regression model
6. Check assumptions
7. Interpret R-squared, coefficient and p-value
8. Generate ROI recommendation
