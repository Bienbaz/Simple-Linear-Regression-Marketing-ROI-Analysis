# Simple Linear Regression – Marketing ROI Analysis

## Project Overview
This project analyzes the relationship between marketing expenditure and sales using Simple Linear Regression.

## Objectives
- Clean and explore data
- Identify strongest marketing predictor
- Build OLS regression model
- Validate assumptions
- Interpret business impact

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels

## Installation

pip install pandas numpy matplotlib seaborn statsmodels scipy

## Files
- regression_analysis.ipynb
- marketing_and_sales_data_evaluate_lr.csv

## Results
The most influential marketing channel was selected based on correlation with sales and evaluated using OLS regression.
